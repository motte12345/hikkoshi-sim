# 引越し費用シミュレーター 仕様書

## サイト概要

- **サイト名**: 引越し費用シミュレーター（仮）
- **目的**: 引越し費用の概算を出し、正確な見積もりへの導線を作る
- **収益モデル**: AdSense + 引越し一括見積もりサイトへのアフィリエイト（高単価ジャンル）
- **技術構成**: React静的サイト（Vite）

## インフラ・運用構成

### ホスティング・デプロイ

- Cloudflare Pages
- GitHubリポジトリと連携し、mainブランチへのpushで自動デプロイ
- プレビューデプロイ：PRごとにプレビューURLが生成される設定にする

### Google連携

- **AdSense**: `<head>`にスクリプトタグ埋め込み。広告ユニットは計算結果の下とサイドバー（PC時）。自動広告も有効化、フォーム内は除外
- **GA4**: gtag.js埋め込み。計算実行をカスタムイベントとして送信
- **Search Console**: ドメイン所有権はCloudflare DNS TXTレコードで確認。サイトマップ登録

### 環境変数管理

- GA4測定ID、AdSenseパブリッシャーIDはCloudflare Pagesの環境変数に入れ、ビルド時注入
- `.env.example`をリポジトリに含める

## 計算ツール一覧

### 1. 引越し費用シミュレーター（メイン） `/estimate`

**入力:**
- 現在の間取り（1R〜5LDK、ドロップダウン）
- 荷物量の目安（少なめ / 普通 / 多め）— 間取りからデフォルト値を自動設定、手動調整可
- 移動距離（km直接入力 or 都道府県→都道府県で概算距離を自動算出）
- 引越し時期（月を選択。3〜4月は繁忙期係数1.3〜1.5）
- 曜日（平日 / 土日祝。土日祝は1.1〜1.2倍）
- 時間帯（午前便 / 午後便 / フリー便。フリーが最安）
- オプション（エアコン脱着、ピアノ、洗濯機設置、不用品処分など。チェックボックス）

**出力:**
- 概算費用（レンジで表示：例「85,000円〜120,000円」）
- 内訳（基本料金、距離加算、時期加算、オプション）
- 「正確な見積もりを取る」ボタン（アフィリエイトリンク）

### 2. 単身パック比較 `/tanshin`

**入力:**
- 荷物量（段ボール数、大型家具の有無）
- 移動距離

**出力:**
- 主要単身パックの概算費用比較（日通、ヤマト、サカイ等の一般的な料金帯）
- 単身パックに収まるか収まらないかの判定
- 収まらない場合は通常プランの概算も表示

### 3. 引越し初期費用トータル計算 `/shoki-hiyo`

**入力:**
- 新居の家賃
- 敷金・礼金（月数）
- 仲介手数料（月数）
- 火災保険料
- 鍵交換費用
- 引越し費用（ツール1の結果を引き継げる）

**出力:**
- 初期費用の合計
- 内訳の表
- 「引越しだけでなく、これだけかかる」を可視化

### 4. 不用品処分コスト計算 `/fuyohin`

**入力:** 品目ごとにチェック（冷蔵庫、洗濯機、テレビ、エアコン、ソファ、ベッド等）

**出力:**
- 自治体の粗大ゴミの目安費用
- リサイクル家電の法定リサイクル料
- 「買取に出した場合」の目安（差額を表示）

## 料金計算ロジック

- DBは使わず、料金テーブルをJSONで管理
- 概算はレンジで出す（計算値の±15%程度）
- 精度を上げすぎない（「実際と違った」クレーム防止）、あくまで目安であることを強調

### 料金テーブル構造イメージ

```json
{
  "basePriceByLayout": {
    "1R": { "少なめ": 30000, "普通": 38000, "多め": 45000 },
    "1LDK": { "少なめ": 45000, "普通": 55000, "多め": 70000 }
  },
  "distanceSurcharge": [
    { "maxKm": 15, "price": 0 },
    { "maxKm": 50, "price": 10000 },
    { "maxKm": 200, "price": 25000 },
    { "maxKm": 500, "price": 45000 },
    { "maxKmOver": true, "price": 70000 }
  ],
  "seasonMultiplier": {
    "1": 1.0, "2": 1.1, "3": 1.5, "4": 1.3, "5": 1.0
  },
  "options": {
    "エアコン脱着": 15000,
    "ピアノ": 30000
  }
}
```

### 都道府県間の距離

- 47×47の距離マトリクスをJSONで持つ
- 都道府県庁所在地間の直線距離×1.3（道路距離の概算）で事前計算
- 約50KB程度

## ページ構成

### ツールページ共通レイアウト

```
[ツール名・H1]
[説明文（2〜3行）]
[計算フォーム]
  - ステップ形式（STEP1 → STEP2 → STEP3 → STEP4）
[計算結果]

  - 内訳の表
```

## SEO対策

- 各ページにmeta description、OGP、構造化データ（FAQSchema）
- サイトマップをSearch Consoleに登録
- 検索意図に合わせたH1・見出し構成（「引越し 費用 計算」「引越し 相場 一人暮らし」等）
- 補足コンテンツでロングテールを拾う

## 差別化ポイント

1. ステップ形式で入力しやすい（既存サイトはフォーム詰め込みで見づらい）
2. 繁忙期・曜日・時間帯の料金差を明示
3. 引越し費用だけでなく初期費用トータルを出せる
4. 不用品処分のコストも含めてトータルで考えられる

## 免責事項

- 概算であり、実際の費用は引越し業者の見積もりにより異なる
- 特定の業者を推奨するものではない
- オプション料金は業者により大きく異なる
